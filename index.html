Mô phỏng thuật toán tìm kiếm tuyến tính

function linearSearch(array, value) {

for (let i = 0; i < array.length; i++) {

if (array[i] === value) {

return i; // Giá trị được tìm thấy

}

}

return -1; // Giá trị không được tìm thấy

}

Hướng dẫn sử dụng

  • Nhập danh sách giá trị các số vào ô "Điền các số". Các số cần được phân tách bởi dấu phẩy.
  • Nhập giá trị cần tìm vào ô "Số cần tìm".
  • Ấn nút "Start" để bắt đầu quá trình tìm kiếm.
  • Quá trình tìm kiếm sẽ được mô phỏng trực quan trên màn hình, cho phép quan sát từng bước của thuật toán.

Giải thích thuật toán tìm kiếm tuần tự

Linear Search là một thuật toán tìm kiếm đơn giản, trong đó danh sách được duyệt tuần tự để tìm giá trị cần tìm. Tại mỗi bước, thuật toán so sánh giá trị cần tìm với giá trị tại vị trí hiện tại trong danh sách.

Nếu giá trị được tìm thấy, thuật toán trả về vị trí của giá trị đó trong danh sách. Nếu không tìm thấy giá trị sau khi đã kiểm tra toàn bộ danh sách, thuật toán trả về -1.
